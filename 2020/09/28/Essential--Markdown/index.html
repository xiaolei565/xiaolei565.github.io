1. 标题

#标题 ———— 一级标题
##标题 ———— 二级标题
###标题 ———— 三级标题
####标题 ———— 四级标题
#####标题 ———— 五级标题
######标题 ———— 六级标题

2. 字体

*文字* ———— 斜体
**文字** ———— 粗体 
***文字*** ———— 粗斜体
~~文字~~ ———— 删除线

3. 列表

-文字1
-文字2 ———— 无序列表（“+”也可以，列表嵌套，在上一级和下一级之间敲三个空格即可）

1. 文字1
2. 文字2 ———— 有序列表（数字.后面需要加一个backspace）

4. 链接

[显示文本](链接地址) ———— 文字链接（括号中地址支持网络地址）

![文本](图片链接地址) ———— 图片链接（文本域可空）

5. 引用

> 文本 ———— 引用（可以无限叠加） 
''' 
    代码段
''' ———— 代码块引用
' 代码 ' ———— 单行代码引用

6. 分隔

***或--- ———— 表示分割线

7. 表格

表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。
有一个就行，为了对齐，多加了几个
文字默认居左
两边加：表示文字居中
右边加：表示文字居右
